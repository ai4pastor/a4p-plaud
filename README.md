# Plaud (a4p)

Plaud 녹음과 전사록을 옵시디언으로 가져오는 플러그인입니다. **읽기 전용**으로 동작하며, 메인 vault가 무거워지지 않도록 텍스트(트랜스크립트·요약)만 노트로 저장하고 오디오는 사이드패널에서 스트리밍으로 재생합니다.

## 주요 기능 (예정)

- 옵시디언 사이드패널에서 Plaud 녹음 검색 (날짜·키워드)
- 트랜스크립트·AI 요약을 옵시디언 노트로 임포트 (수동)
- 사이드패널 스트리밍 플레이어 (재생 직전 presigned URL 자동 갱신)
- 트랜스크립트 ↔ 오디오 양방향 점프, "현재 위치 인용" 삽입
- 선택적 오프라인 캐시 (특정 녹음만 다운로드)

## 설계 원칙

- vault 안에는 텍스트만 — 오디오 mp3는 vault에 저장하지 않음
- Plaud → 옵시디언 단방향 (Plaud 데이터를 수정하지 않음)
- 토큰은 electron `safeStorage`로 암호화 저장
- 한국어 UX
